# プロフィール編集

ログインユーザは `/profile` ページから以下の情報を編集できる。

## 表示名（display_name）

- テキスト入力フォームで設定・変更できる
- 最大 50 文字。空欄にすると未設定扱いになる
- 将来的にコメント機能などでユーザ名として表示する基盤となる

## アイコン画像

- PNG / JPEG 形式、最大 1MB のファイルをアップロードできる
- アップロードした画像は MinIO バケット（`minitube`）の `user-icons/{user_id}.{ext}` に保存される
- 既存のアイコンがある場合は上書きされる
- プロフィールページでは CSS により丸く表示される

### MinIO 上のアイコン保存先

```
user-icons/
└── {user_id}.png   ← PNG の場合
└── {user_id}.jpg   ← JPEG の場合
```

## パスワード変更

- 現在のパスワード・新しいパスワード・確認用パスワードの 3 項目を入力する
- 現在のパスワードが一致しない場合はエラーを返す
- 新しいパスワードは 8 文字以上である必要がある
- 成功時はログイン状態を維持したままパスワードが更新される

## API 仕様

| エンドポイント | メソッド | 説明 |
|--------------|---------|------|
| `/profile` | GET | プロフィール表示・編集ページ |
| `/api/profile/update` | POST | 表示名・アイコン画像の更新 |
| `/api/profile/password` | POST | パスワードの変更 |

### `/api/profile/update` のフォームパラメータ

| パラメータ | 型 | 必須 | 説明 |
|-----------|---|------|------|
| `display_name` | string | 任意 | 表示名（最大 50 文字） |
| `icon` | file | 任意 | アイコン画像（PNG / JPEG、最大 1MB） |

### `/api/profile/password` のフォームパラメータ

| パラメータ | 型 | 必須 | 説明 |
|-----------|---|------|------|
| `current_password` | string | 必須 | 現在のパスワード |
| `new_password` | string | 必須 | 新しいパスワード（8 文字以上） |
| `confirm_password` | string | 必須 | 新しいパスワード（確認用） |

## データベース変更

`users` テーブルに以下のカラムを追加（マイグレーション: `5e6f7g8h9i0j`）:

| カラム | 型 | NULL 許可 | 説明 |
|-------|---|----------|------|
| `display_name` | VARCHAR(50) | ✓ | ユーザの表示名 |
| `icon_path` | TEXT | ✓ | MinIO に保存されたアイコン画像のオブジェクトキー |
